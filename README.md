# ポートフォリオ（My Portfolio）

私の制作実績やスキルをまとめたポートフォリオサイトです。
HTML、CSS、JavaScriptを使用して、シンプルで機械的なデザインを目指して構築しました。

## 🚀 公開URL
- [ポートフォリオサイトはこちら](https://warahagi.github.io/portfolio/)

## 🛠️ 主な制作実績
リポジトリ内に含まれる主な作品の紹介です。
- **クイックフィックツール (quick-fix-tool)**: よく使う定型文やコード fragments を瞬時にコピーできる snippet 管理ツール。
- **点滴計算ツール (iv-drip-calc)**: 医療現場などの計算をサポートするWebツール。
- **サンデーサイレンス3Dシミュレーター (suzuka-3d-sim)**: 3D要素を取り入れたシミュレーションツール。
- **ポートフォリオ (pdf)**: これまでの20年以上のデザイン・印刷キャリアを凝縮した総合ポートフォリオ。

## 💻 使用技術
- HTML5 / CSS3
